# AI-Keycap
I made a 3D-Printable custom keycap that has "AI" printed on it, as I will be binding that spot on my keyboard to my AI tools. Designed for Cherry MX switches on the Everest 60 Mechanical Keyboard.
# Why I Made This
I made it because my right Alt key is already binded to opening up my ai tools, so now I have a keycap that physically says AI so it's a lot easier to remember that the binding exists.
