# AI-Keycap
I made a 3D-Printable custom keycap set with a modernized start key, a music key, and an ai tools key. Designed for Cherry MX switches on the Everest 60 Mechanical Keyboard. In the image of the irl keycaps, they are already on the keyboard. The three original keys are placed under them.
# Why I Made This
I made it because my right Alt key is already binded to opening up my ai tools, so now I have a keycap that physically says AI so it's a lot easier to remember that the binding exists. I also wanted to redesign my start key, as well as adding a key to open spotify.
